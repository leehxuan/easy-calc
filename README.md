# 随算（EasyCalc）

### 基本信息
- 中文名随算，英文名 <i>EasyCalc</i>。是一个轻量快捷的终端计算器，由 <i>leehxuan</i> 开发，软件开源于 [Github](https://github,com/leehxuan/easy-calc/)。

### 使用指南
1. 运行可执行文件；
2. 输入命令（所有可用命令请见[命令列表](#命令列表)）；
3. 按 `ALT + F4` 或使用命令 `q` 退出程序。

### 命令列表
|编号|命令|作用|参数|
|:-:|:-:|:-:|-|
|A001|`c [a]`|计算|`a` 表达式|
|A002|`ch [a*o] [b]`|选择常数|`a` 常数集（默认常数集无需标注，所有常数集请见[常数集列表](#常数集列表)）<br />`b` 常数代码（常数在常数集中的对应代码）|
> 单字母如 `a`、`b` 等为参数，后跟 `*o`表示可选参数。
